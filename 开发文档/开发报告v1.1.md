## 一、项目基本信息总结

| 项目项 | 当前情况 |
|---|---|
| 项目名称 | AI Innovation Agent Studio (智创工坊) |
| 前端框架 | Next.js 15 + React 19 + TypeScript |
| 后端框架 | FastAPI 0.109+ |
| 数据库 | PostgreSQL 16 + pgvector |
| 向量数据库/向量扩展 | pgvector (已配置，embedding 功能部分实现) |
| Redis/队列 | Redis 7 (已配置，未实际使用) |
| 文件存储 | MinIO (已配置，文件上传已实现) |
| UI 组件库 | shadcn/ui + Radix UI + Tailwind CSS |
| Agent 框架 | 自研 Agent Service (基于关键词匹配的 Skill 选择) |
| RAG 是否实现 | 部分实现 - 有文档上传和 chunk 存储，embedding 和向量检索为 mock |
| Memory 是否实现 | 已实现 - 支持 CRUD 和基础检索 |
| Skill Registry 是否实现 | 已实现 - 支持 Skill 列表、启用/禁用 |
| Tool Calling 是否实现 | 部分实现 - 有 Tool 注册和调用记录，但审批流程为 mock |
| Eval 是否实现 | 部分实现 - 有评估记录存储，但评分为 mock |
| Trace/Observability 是否实现 | 未实现 - Langfuse 配置存在但未集成 |
| 当前是否主要依赖 mock data | 是 - Agent 输出、评估分数、计划生成均为 mock |

**额外说明：**
- 当前项目是 monorepo，使用 pnpm workspace
- 项目有 `apps/web`（前端）、`apps/api`（后端）目录
- 项目可以被视为"半成品 Demo"，前端页面完整，后端 API 已实现，但核心 AI 能力为 mock
